DOS HEALTHCHECK SCRIPTS

HEALTH.BAT
  Conservative baseline script for MS-DOS 3.x, 4.x, 5.x and 6.x.
  Creates DOS-REPORT.TXT.

HEALTH6.BAT
  Extended script for MS-DOS 6.x systems.
  Uses MEM /C and MEM /D style checks.
  Creates DOS6-REPORT.TXT.

These scripts are read-only. They collect text output and do not modify
CONFIG.SYS, AUTOEXEC.BAT or system files.
