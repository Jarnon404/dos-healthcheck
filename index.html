DOS Healthcheck

Retro-style system audit and health check toolkit for MS-DOS 3.x, 4.x, 5.x and 6.x machines.

Repository Releases

DOS 3.x / 4.x

Conservative baseline checks using classic commands and syntax.

DOS 5.x

Baseline audit with more useful memory reporting where available.

DOS 6.x

Extended script with detailed memory reporting and DOS 6.x tool checks.

Output

DOS-REPORT.TXT
DOS6-REPORT.TXT

Read-only by design. The scripts collect command output and do not modify CONFIG.SYS or AUTOEXEC.BAT.
